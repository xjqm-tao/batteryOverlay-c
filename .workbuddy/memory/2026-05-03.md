# 2026-05-03 工作日志

## Battery Overlay C++ 项目

### 新增功能：单击切换显示时间模式

为悬浮窗添加了单击切换显示模式的功能：

**功能说明：**
- 单击悬浮窗（不是拖拽）切换显示模式
- 时间模式：第一行 `hh:mm`（小时:分钟），第二行实时秒数
- 再次单击恢复原来的显示（输入法状态 + 电池百分比）

**实现方式：**
- 添加 `AppState::showTime` 原子变量跟踪显示模式
- 添加 `AppState::clickStartX/Y` 记录单击起始位置
- 在 `WM_LBUTTONUP` 中判断移动距离 < 3 像素则为单击，切换 `showTime` 状态
- 在 `WM_CAPTURECHANGED` 中也处理单击（防止窗口外释放鼠标时漏判）
- 修改 `render()` 函数根据 `showTime` 状态渲染不同内容

**修改文件：**
- `main.cpp`：添加状态变量、修改 render()、WM_LBUTTONDOWN、WM_LBUTTONUP、WM_CAPTURECHANGED
- `BatteryOverlayC++开发笔记.md`：记录新功能
